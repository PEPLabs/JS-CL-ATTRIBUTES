Setting Attributes

Recall that elements in the DOM are described in HTML with tags. Tags can have attributes which control various aspects of the element. You can add styling with the "style" attribute, you can add a destination to an anchor (link) tag with the "href" attribute, you can add a reference to an image element with the "src" attribute. In this exercise you will need to display an image by altering an image element's attributes.
